AgentDock is a local-first multi-agent coding console that brings Codex, Claude Code, and OpenCode threads into one desktop workspace for viewing and continuation. It does not replace existing CLIs. Instead, it treats native agent threads as the source of truth and provides local indexing, grouping, and resume operations to make multi-agent workflows more continuous and controllable.

Key points:

- Unified thread management: view, filter, and resume historical threads across three agents
- Local-first: runtime, data, and integrations all stay on the local machine
- Desktop-first experience: built with Tauri + React, currently running in terminal-only mode
- Clear scope: `health_check`, `list_threads`, and `resume_thread` are implemented; cross-agent summary orchestration and a complete mobile loop are still in planning
